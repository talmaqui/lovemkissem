Add to VSCode

To add Lovem Kissem to your VSCode custom snippets, whether you use Visual Studio Code or not:

  1. In VSCode, open your preferences menu
  2. You'll see the User Snippets menu, which you can add a global custom snippet to
  3. Go to Configure User Snippets
  4. In my case, I created a new separate file called lovem, but you can add this snippet to your global snippet files if you want!
  5. After you save it in this format, with a description and prefix, you can type the "love" prefix into a file, hit enter/return and the paragraph will generate!
  6. For further customization options, you can create small, medium, and big sizes of Lovem Kissem!

about

I love romance and romance poetry! In the initial conception of this project, I was going to pull the most common words from romantic comedy scripts, but I find lorem ipsums poetic and poetry just seemed more fun! I pulled romance poetry from poetry websites (with Beautiful Soup) and made a small word-counter python program (nothing special, it works like most of them!) to find the most common words in them, grabbed those words, cleaned the data (removed words like "the" and "a"), and then randomized them! I did want to start it off as romantically as possible, though, so some of my human influence changed the arrangements!
